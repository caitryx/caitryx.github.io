name == ‘main‘的作用

摘要

这句话d的作用相当于是程序的入口，有点类似于Java中的main方法，直接运行java文件便会执行main方法里面的函数及变量，从其他类进行导入创建实例化对象的时候需要自行调用，main方法里面的内容不会执行，但python中没有统一的入口，都是从脚本第一行开始运行。
